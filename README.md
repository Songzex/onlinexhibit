![小说阅读](https://github.com/user-attachments/assets/81e6b014-d8e2-4c3c-9975-76dc74f6559d)# 项目介绍！《狗儿哥漫画APP》单体架构版本

------

## 本项目含有的功能：

1. 登录注册
2. 观看漫画
3. 收听广播剧
4. 评论系统
5. 聊天系统（单聊群聊）
6. 消息通知
7. 收藏点赞订阅功能
8. Feed帖子系统（发布和接收）
9. 添加好友群组（社区功能）
10. 搜索系统

## :项目技术

开发语言：JAVA （jdk1.8）

框架： SpringBoot2.6,	Mybatis  UNiApp  Netty  Websocket

数据库： Mysql  Redis  Mongodb
